# QR Generator JS

Página estática que genera códigos QR en el navegador.

## Uso rápido
1. Abre [`index.html`](index.html) o visita la demo en GitHub Pages.
2. Escribe texto / URL.
3. Pulsa **Generar** → descarga la imagen con clic derecho → *Guardar imagen como…*

## Stack
- [qrcode.js](https://github.com/soldair/node-qrcode) vía CDN
- 100 % cliente (sin backend)
